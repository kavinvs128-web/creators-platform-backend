# MERN Frontend

This is the React frontend for the MERN stack application, built with Vite.

## Getting Started

1. Install dependencies:

   ```
   npm install
   ```

2. Start the development server:

   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```

## Features

- Modern React application
- Fast development with Vite
- ESLint configuration for code quality
- Responsive design

## Technologies

- React
- Vite
- ESLint
